# 💧 Water Intake Tracker

A simple, responsive water intake tracker built as my first web development project.

## Features
- Set a daily water goal (number of glasses)
- Track progress with a live progress bar
- Get a congrats message when you hit your goal
- Progress saves automatically using localStorage — refresh the page and your data stays

## Built With
- HTML5
- CSS3
- JavaScript (vanilla, no frameworks)

## What I Learned
This was my first project, built to practice core web fundamentals: DOM manipulation, event listeners, and browser localStorage.
